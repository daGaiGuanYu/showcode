# 功能与特点
+ 在网页上写代码，随写随看
+ 分享网页代码
+ 超轻量级（html、css、js 加起来不超过 300 行）
+ 没有后端，不保存代码（你分享的代码被放入 url 里，如果你的代码很长，url 自然也会很长）
